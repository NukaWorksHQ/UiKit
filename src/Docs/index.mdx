# UiKit Documentation

A modern, accessible React component library built with TypeScript and styled-components.

## Quick Start

```bash
npm install @uikit/core
# or
yarn add @uikit/core
```

## Basic Example

```tsx live
function Example() {
  return (
    <div style={{ display: "flex", gap: "8px" }}>
      <Button>Default</Button>
      <Button color="Primary">Primary</Button>
      <Button disabled>Disabled</Button>
    </div>
  );
}
```

## Design Principles

1. **Accessibility First**

```tsx live
<Button aria-label="Add item" onClick={() => alert("Button clicked!")}>
  Add Item
</Button>
```

2. **Type Safety**

```tsx
// All components are fully typed
interface ButtonProps {
  color?: "Default" | "Primary";
  size?: "small" | "medium" | "large";
  disabled?: boolean;
  children: React.ReactNode;
}
```

3. **Customizable**

```tsx live
<div style={{ display: "flex", gap: "8px", flexDirection: "column" }}>
  <Button style={{ backgroundColor: "#4CAF50", color: "white" }}>
    Custom Green
  </Button>
  <Button style={{ backgroundColor: "#2196F3", color: "white" }}>
    Custom Blue
  </Button>
</div>
```

4. **Modern React Patterns**

```tsx
// Using hooks for state management
function ControlledInput() {
  const [value, setValue] = React.useState("");
  return (
    <TextField
      value={value}
      onChange={(e) => setValue(e.target.value)}
      label="Controlled Input"
    />
  );
}
```

## Form Components

Example of a complete form:

```tsx live
function FormExample() {
  const [submitted, setSubmitted] = React.useState(false);

  return (
    <form
      style={{
        display: "flex",
        flexDirection: "column",
        gap: "16px",
        maxWidth: "300px",
      }}
      onSubmit={(e) => {
        e.preventDefault();
        setSubmitted(true);
        setTimeout(() => setSubmitted(false), 2000);
      }}
    >
      <TextField
        label="Email"
        type="email"
        required
        placeholder="Enter your email"
      />

      <TextField
        label="Password"
        type="password"
        required
        placeholder="Enter password"
      />

      <Checkbox label="Remember me" />

      <Button color="Primary" type="submit" style={{ marginTop: "8px" }}>
        {submitted ? "Submitted!" : "Submit"}
      </Button>
    </form>
  );
}
```

## Best Practices

### Component Composition

```tsx live
function CompositionExample() {
  return (
    <div style={{ display: "flex", flexDirection: "column", gap: "8px" }}>
      <Text size={16} weight="bold">
        User Settings
      </Text>

      <div style={{ display: "flex", alignItems: "center", gap: "8px" }}>
        <Switch label="Dark Mode" />
        <Text size={14} color="gray">
          (Changes app theme)
        </Text>
      </div>

      <div style={{ display: "flex", alignItems: "center", gap: "8px" }}>
        <Switch label="Notifications" />
        <Text size={14} color="gray">
          (Email updates)
        </Text>
      </div>
    </div>
  );
}
```

### Accessibility

```tsx live
function AccessibleForm() {
  return (
    <form role="search" aria-label="Search documentation">
      <div style={{ display: "flex", gap: "8px" }}>
        <TextField
          label="Search"
          placeholder="Search docs..."
          aria-label="Search documentation"
        />
        <Button type="submit" aria-label="Submit search">
          Search
        </Button>
      </div>
    </form>
  );
}
```
