## API Reference

### Radio Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `color` | `"Default" \| "Primary" \| "Success" \| "Warning" \| "Alert" \| "Disabled"` | `"Default"` | Color variant |
| `label` | `string` | - | Radio button label |
| `checked` | `boolean` | `false` | Controlled checked state |
| `onChange` | `(event: React.ChangeEvent<HTMLInputElement>) => void` | - | Change handler |
| `name` | `string` | - | Input name attribute |
| `value` | `string` | - | Radio button value |
| `disabled` | `boolean` | `false` | Disables the radio button |

### Radio.Group Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `name` | `string` | - | Group name for radio buttons |
| `value` | `string` | - | Selected radio value |
| `onChange` | `(value: string) => void` | - | Change handler |
| `direction` | `"horizontal" \| "vertical"` | `"vertical"` | Layout direction |
| `children` | `ReactNode` | - | Radio button components |

## Best Practices

### Do's
- Use for mutually exclusive options
- Group related radio buttons
- Provide clear, concise labels
- Use appropriate spacing between options

### Don'ts
- Don't use for multiple selections (use checkboxes instead)
- Avoid using a single radio button
- Don't use for binary choices (use switch or checkbox instead)
- Avoid nesting radio groups

## Examples

### Controlled Radio Group

```tsx live
function ControlledRadioExample() {
  const [selected, setSelected] = React.useState('email');

  return (
    <AppActivity>
      <div style={{ display: 'flex', flexDirection: 'column', gap: '16px' }}>
        <div>Preferred Contact Method</div>
        <Radio.Group
          name="contact"
          value={selected}
          onChange={setSelected}
        >
          <Radio value="email" label="Email" color="Primary" />
          <Radio value="phone" label="Phone" color="Primary" />
          <Radio value="mail" label="Mail" color="Primary" />
        </Radio.Group>
        <div style={{ fontSize: '14px', color: 'gray' }}>
          Selected: {selected}
        </div>
      </div>
    </AppActivity>
  );
}
```
